# Grown Up Boy

*A Game Boy emulator built in `C++` for fun and learning.*

This emulator is a personal project focused on learning and experimentation. It is not guaranteed to be the most efficient or the most accurate implementation.


## Building

The project is currently undergoing a major architectural overhaul.

I decided to hit the brakes on new features to refactor the core codebase and clean up the old mess. Check back soon for a more stable (and much cleaner) release!
